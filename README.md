# CTOD Demo App

Demo-first prototype for an end-to-end CTOD literature curation workbench.

## Features
- Create curation project
- Configure schema fields and automation lanes
- Mock extraction run
- Review extracted values with evidence and confidence
- Edit / accept / reject values
- Publish summary with learning-loop metrics

## Run locally
```bash
npm install
npm run dev
```

## Deploy
- Push to GitHub
- Import repo into Vercel or Render
- Build command: `npm run build`
- Start command: `npm run start`

## Notes
This is a demo-first UI prototype using browser localStorage for persistence.
